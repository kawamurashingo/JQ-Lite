jq-lite

Pure Perlで実装された軽量な jq 互換エンジン。
制約の多い環境でも、JSON / YAML を確実に扱うための道具です。

日本の道具のように
設計方針を見る

主な特徴

Pure Perl

Perlが動く環境ならどこでも。

jq互換

jqと同じ感覚で使える。

安定したCLI

互換性を壊さない設計。

Alpine / CI / Legacy

Alpine Linux

apk add jq-lite

CI / Docker

jq-lite -e '.ok == true' result.json

Legacy / Offline

cpanm JQ::Lite

使い方（CLI）

# JSON から値を取り出す
jq-lite '.users[].name' users.json

# 条件に合う要素だけを抽出
jq-lite '.users[] | select(.age > 25)' users.json

# CI でのチェック（失敗時は非0終了）
jq-lite -e '.ok == true' result.json
