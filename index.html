jq-lite

日本の道具のように、静かに、確実に。

jq-lite は、pure Perl で実装された 軽量な jq 互換 JSON プロセッサです。
新しいバイナリを入れられない環境でも、 すでにそこにある Perl を使って JSON を扱えます。

For English readers: English overview

なぜ jq-lite なのか

世の中には、どうしても環境を変えられない現場があります。
古い UNIX、厳しいセキュリティ制約、最小構成のコンテナ。

jq-lite は、そうした現場で 「それでも仕事を進める」 ための道具です。

想定される利用シーン

  • jq をインストールできないサーバ
  • Perl が標準で入っているレガシー環境
  • Alpine / BusyBox などの最小コンテナ
  • 工場設備・IoT ゲートウェイ
