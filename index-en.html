From setup to day-to-day use

JQ::Lite helps operations teams who cannot spare time for complex environments. Automate JSON processing with just three steps.

  • 1

    Deploy the files

    Install from CPAN with cpanm or place the single file on your server.

  • 2

    Write filters

    Use jq-style expressions to extract and transform JSON. Reuse the knowledge you already have.

  • 3

    Integrate with scripts

    Require it as a Perl module or pipe data through the CLI‚Äîboth work seamlessly.

Config Snapshot

# Extract specific users
$ jq-lite users.json '.users[] | select(.role == "admin") | {name, last_login}'

# Use from a Perl script
use JQ::Lite qw/jq/;
my $json = decode_json($raw);
my $admins = jq('.users[] | select(.role == "admin")', $json);
