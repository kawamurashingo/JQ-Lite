A lightweight jq-compatible engine written in pure Perl.

Highlights

jq-compatible

CLI & library

Use it everywhere.

Installation

From CPAN

cpanm JQ::Lite

Alpine Linux

apk add jq-lite

Manual install (CPAN tarball)

ver=$(curl -s https://fastapi.metacpan.org/v1/release/JQ-Lite \
  | perl -MJSON::PP -0777 -ne 'print decode_json($_)->{version}')

curl -sSfL http://cpan.metacpan.org/authors/id/S/SH/SHINGO/JQ-Lite-$ver.tar.gz -o JQ-Lite-$ver.tar.gz

tar xzf JQ-Lite-$ver.tar.gz
cd JQ-Lite-$ver

perl Makefile.PL INSTALL_BASE=$HOME/.local
make
make test
make install

export PATH="$HOME/.local/bin:$PATH"
export PERL5LIB="$HOME/.local/lib/perl5:$PERL5LIB"
